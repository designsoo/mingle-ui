# Welcome to the Mingle Component library

---

mingle-ui is a powerful design system built with Tailwind CSS and Storybook.\
It’s designed to help developers and designers create consistent, beautiful interfaces effortlessly.\
Our library offers a variety of reusable and customizable components that follow best practices, ensuring a cohesive look and feel across all your projects.

# Install

---

Install mingle-ui into your project using NPM:

```bash
npm install mingle-ui
```

If you are already using mingle-ui and want to update to the latest version, use the following command:

```bash
npm install mingle-ui@latest
```

# Usage

---

Here’s a basic example of how to use the PrimaryButton component in your project:

```tsx
import { PrimaryButton } from 'mingle-ui';

const Component = () => {
  retrun(
    <>
      <PrimaryButton type='submit' disabled={isDisabled} onClick={onButtonClick}>
        Submit
      </PrimaryButton>
      <PrimaryButton size='sm' variant='stroke' disabled={isDisabled} onClick={onButtonClick}>
        Cancel
      </PrimaryButton>
      <PrimaryButton size='lg' variant='destructive' disabled={isDisabled} onClick={onButtonClick}>
        Delete
      </PrimaryButton>
    </>,
  );
};
```
