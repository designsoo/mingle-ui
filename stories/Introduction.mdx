# Welcome to the Mingle Component library

---

mingle-ui is a comprehensive design system built with Tailwind CSS and Storybook, designed to empower developers and designers to build consistent and beautiful interfaces efficiently. mingle-ui provides a set of reusable, customizable components that follow best practices for design and development, ensuring consistency across your projects.

# Install

---

Install mingle-ui into your project using NPM:

```bash
npm install mingle-ui
```

If you are already using mingle-ui and want to update to the latest version, use the following command:

```bash
npm install mingle-ui@latest
```
